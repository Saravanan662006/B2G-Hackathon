About Signify

Signify is a gesture-to-speech system designed to empower communication for individuals with speech impairments. It uses flex sensors and machine learning to translate hand signs into spoken words.

Users can customize gestures to represent personal vocabulary like names, places, or phrases. This makes Signify adaptable across languages and cultures.

Tech Stack: Arduino/Raspberry Pi, Python (scikit-learn/TensorFlow), Flask, HTML/CSS/JavaScript, Web Speech API.
